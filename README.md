# 🎓 Etec Zona Leste - Site Institucional

## 📝 Sobre o Projeto

Este projeto é um site institucional desenvolvido para a **Etec Zona Leste**, com o objetivo de apresentar a escola, seus cursos, eventos e fornecer um canal de comunicação com os alunos e comunidade.
Tudo isto de forma fictícia, claro. Sou uma estudante dessa escola, e esse projeto de site institucional foi a pedido do professor de Programação Web 2, Marcello Collado.
O read.me foi estruturado com a ajuda de inteligência artificial, assim como algumas funções dos sites devido ao limite de tempo de entrega do site.
Apesar do site ser simples, foi focado o back-end do site, nesse caso, o php, o modelo simples permitiu que a criadora do site aproveitasse o tempo para estudar mais afundo o php para que pudesse trazer um melhor resultado conforme seus estudos.

## 🐘 PHP - Formulário de Contato

### Arquivo: `formulario.php`

O PHP foi utilizado para processar o formulário de contato de forma **fictícia**, ou seja, uma simulação, demonstrando os conceitos estudados em aula.

#### Funcionalidades do PHP:

| Função | Descrição |
|--------|-----------|
| **Captura de dados** | Recebe os dados via `$_POST` |
| **Validação de campos** | Verifica se nome, e-mail e mensagem foram preenchidos |
| **Validação de e-mail** | Usa `filter_var()` para validar formato do e-mail |
| **Mensagens de erro** | Exibe mensagens específicas para cada erro |
| **Mensagem de sucesso** | Simula o envio com mensagem de confirmação |
| **Redirecionamento** | Volta para a página de contato após exibir mensagem |

#### Código principal do PHP:

```php
<?php
// Captura os dados do formulário
$nome = trim($_POST['nome'] ?? '');
$email = trim($_POST['email'] ?? '');
$mensagem = trim($_POST['mensagem'] ?? '');

// Validação dos campos
if (empty($nome)) {
    $mensagem = "Por favor, informe seu nome.";
    $tipo = "erro";
} 
elseif (empty($email)) {
    $mensagem = "Por favor, informe seu e-mail.";
    $tipo = "erro";
} 
elseif (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
    $mensagem = "Por favor, informe um e-mail válido.";
    $tipo = "erro";
} 
elseif (empty($mensagem)) {
    $mensagem = "Por favor, escreva sua mensagem.";
    $tipo = "erro";
} 
else {
    $mensagem = "Mensagem enviada com sucesso!";
    $tipo = "sucesso";
}
?>
